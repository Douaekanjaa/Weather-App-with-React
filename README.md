# ☁️ Weather App ☀️

A simple React app that fetches and displays weather information using the OpenWeatherMap API.

## Features

- 🌍 Display current weather conditions, temperature, and additional information for a specified city.
- 🌡️ Toggle between Fahrenheit and Celsius temperature units.
- 🔍 Dynamically update weather information based on user input.

## Demo

Check out the live demo: [Weather App Demo](https://douaekanjaa.github.io/Weather-App-with-React/)

## 🌐 Usage
- Enter the City Name:
Enter the desired city name in the input field.

- Click "Search":
Click the "Search" button to fetch and display weather information for the specified city.

- Toggle Temperature Units:
Toggle between Fahrenheit and Celsius using the temperature unit button.
